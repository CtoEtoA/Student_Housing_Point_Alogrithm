# Simple Java Scanner Program

This program reads a line of input from the terminal using `Scanner(System.in)` and prints it back.

## Usage

1. Compile the program:
   ```
   javac Main.java
   ```

2. Run the program:
   ```
   java Main
   ```

3. Enter some text when prompted and press Enter.

## Example

```
Enter something:
Hello, World!
You entered: Hello, World!
```

## Troubleshooting

- Ensure you have Java installed (`java -version`).
- If compilation fails, check for syntax errors in `Main.java`.
